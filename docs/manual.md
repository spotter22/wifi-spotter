# wifi-spotter manual
* wifi-spotter is project which contains multiple different utils for testing and recroding the captive-portal networks.


## Installation:

**Before installing:**
* Installing wifi-spotter will require you to root your device, unless you know what you are doing you shouldn't continue installing.
we are not responsiable for any data lose or currupt that may happen to your device.

**Requirements:**
* A device with root access is required.
* Termux app.
* MACsposed app.

**How to install:**
* Installing wifi-spotter is simple as copying and pasting this one liner script into your terminal:
```Bash
curl -sL "https://raw.githubusercontent.com/spotter22/wifi-spotter/refs/heads/main/install.sh" | bash -s -- --install-latest
```
* The script will download some of build tools that are required to compile required utils and other neccssary utils so it's may take a little time.
